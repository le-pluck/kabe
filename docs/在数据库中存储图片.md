# 在数据库中存储图片

+ 需求：
    + 用户头像
    + 帖子封面


## DB 数据类型


| 类型        | 字节数                 | 近似大小 |
| :--------- | :-------------------- | :------ |
| TINYTEXT   | 256 bytes             |         |
| TEXT       | 65,535 bytes          | ~64kb   |
| MEDIUMTEXT | 16,777,215 bytes      | ~16MB   |
| LONGTEXT   | 4,294,967,295 bytes	 | ~4GB    |

将图片转 Base64 格式，以可读字符串的形式存储，则选择 `mediumtext` 是比较合适的。

受 Base64 编码方式影响，16 MB 的空间可以存储最大图片大小约 12 MB；
