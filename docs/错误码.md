# 错误码

设计依据：[HTTP 响应状态码 - HTTP | MDN](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Status)

| status code | indicates    | message        |
| :---------- | :----------- | :------------- |
| 200         | OK           | 请求成功        |
| 201         | Created      |                |
| 202         | Accepted     |                |
|             |              |                |
| 401         | Unauthorized | 身份验证凭证过期 |
|             |              |                |
